# Folder Structure

```text
my-app/
├── docs/
├── public/
├── src/
├── index.html
├── package.json
├── package-lock.json
├── tsconfig.json
└── vite.config.ts
```

## `src/`

Purpose: React application source.

Key files:

- `src/main.tsx`: mounts the React app.
- `src/App.tsx`: portfolio intro component.
- `src/App.css`: component styling.
- `src/index.css`: global styling.

## `public/`

Purpose: static assets served from the site root.

Key files:

- `favicon.svg`
- `icons.svg`

## `docs/`

Purpose: project documentation for future maintainers and agents.

Key files:

- `architecture.md`
- `folder-structure.md`
- `setup.md`

## Generated Folders

- `node_modules/`: installed dependencies.
- `dist/`: production build output generated by `npm run build`.
