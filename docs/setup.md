# Setup

## Prerequisites

- Node.js. Current validated version: `20.11.0`.
- npm.

## Install

```bash
npm install
```

## Develop

```bash
npm run dev
```

Open the local URL printed by Vite.

## Lint

```bash
npm run lint
```

## Build

```bash
npm run build
```

The build output is written to `dist/`.

## Notes

Vite is pinned to `6.4.3` because Vite 7+ and Vite 8 require Node `20.19+`. Upgrade Node before upgrading to the latest Vite line.
